NavRush iOS SDK

NavRush Service provides unique way to track devices utilising all available sources from each of supported device

Features

  • [X] Low Energy Location Tracking
  • [X] GPS High energy Location Tracking
  • [X] Bluetooth Beacons (iBeacons) Ranging
  • [X] Motion Detection - stationary, driving etc.
  • [X] Caching Data & Battery preservation
  • [X] Background & Offline Mode
  • [X] Server-Side Configuration
  • [X] Complete Documentation

Requirements

  • iOS 10.0+
  • NavRush Service Licence (Get Licence Key Here)

Installation

NavRush SDK can be integrated either using dependency manager cocoa-pods or via manual integration.

Cocoa-Pods

Example Pod File

platform :ios, '9.0'
use_frameworks!

target 'MyApp' do
pod 'NavRushSDK-iOS', '~> 1.0'
end

Run pod install
